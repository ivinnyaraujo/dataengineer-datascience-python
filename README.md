# 🧠 Data Engineering | Data Science | Python | R | Microsoft Fabric | CI/CD

## Overview

This repository serves as a personal portfolio and content hub for data engineering and data science workflows. It showcases practical implementations across machine learning, data pipeline automation, and semantic modeling using Python, R, and Microsoft Fabric. The content is not tied to a single project, but rather reflects a curated collection of reusable components, tutorials, and exploratory notebooks.
